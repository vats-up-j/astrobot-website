# Skill: Keyword Opportunities

Identify high-traffic Indian astrology search queries that HoroTalk isn't targeting yet.

## When to trigger
- Strategy agent is planning the content roadmap
- User asks "what should we build next?" or "what content should we create?"
- Monthly planning session

## Steps

1. **Inventory current pages** — List every page and the primary keyword it targets:
   - Homepage → "vedic astrology whatsapp"
   - Panchang → "aaj ka panchang", "today panchang"
   - Rashi Calculator → "mera rashi kya hai", "rashi by name"
   - Features → "ai astrology chatbot"
   - Pricing → "astrology consultation price"
   - (Future sign pages → "[sign] rashifal today")
   - (Future blog posts → long-tail queries)

2. **Identify gaps** — High-volume Indian astrology queries NOT covered:

   **Daily recurring** (people search these EVERY day):
   - "aaj ka rashifal [sign]" (daily horoscope per sign)
   - "rahu kaal today [city]" (city-specific timing)
   - "choghadiya today" (auspicious time slots)
   - "shubh muhurat today" (auspicious timings)

   **Life milestone** (high intent, people search at key moments):
   - "kundli matching for marriage" / "gun milan"
   - "mangal dosha check"
   - "sade sati check" / "shani sade sati [sign] 2026"
   - "baby name by rashi" / "rashi se naam"

   **Evergreen informational** (people search anytime):
   - "[sign] personality traits" / "[sign] characteristics"
   - "[planet] in [sign] effects"
   - "vedic astrology vs western astrology"
   - "what is [nakshatra] nakshatra"
   - "remedies for [planet] dosha"

   **Seasonal spikes** (high volume around festivals/events):
   - "navratri 2026 dates"
   - "diwali muhurat 2026"
   - "griha pravesh muhurat 2026"
   - "vivah muhurat 2026"

3. **Score each opportunity** using the Strategy framework:
   - Traffic potential (0-5)
   - Conversion potential (0-5)
   - Effort (0-5, inverted)
   - Differentiation (0-5)

4. **Output format**:
   ```
   ## Keyword Opportunities — [date]

   ### Top 5 recommendations (build next)
   1. [keyword] — Score: X/5
      Page type: [tool/blog/sign page]
      Est. effort: [low/medium/high]
      Conversion path: [how it leads to WhatsApp]

   ### Backlog (build later)
   - [keyword list with scores]
   ```
